# Проверка публичной сборки

`verify-public.mjs` — единый privacy/demo/manifest gate для локального запуска и
GitHub Actions. У скрипта нет npm-зависимостей: нужен только Node.js 20 или новее.

```bash
node .quality/verify-public.mjs
```

Проверяются:

- отсутствие рабочего `roadmap-data*.json` вне папки схем;
- отсутствие известных рабочих маркеров `RHEAD-`, `WEBADM-`, `rambler-co.ru` в
  открытых текстовых файлах;
- шифрование всех файлов в `private/`, кроме страницы-индекса;
- `noindex` у закрытого раздела;
- SHA-256, размеры и подключения видео, постера и 82 скриншотов;
- пары светлой и тёмной темы для каждой функции;
- базовая структура HTML и валидность JSON-манифестов/схемы;
- отсутствие прежних `demo.mp4` и `demo.webm`.
- отсутствие устаревшего `features.png` и ссылок на него.

Если меняется видео или скриншот, сначала обновляется соответствующий манифест.
Ослаблять проверку под неподготовленный файл нельзя: публичный артефакт сначала
обезличивается, затем получает новый hash.
