# Дорожная карта продукта — v13.3

Наглядный план продукта в одном офлайн-файле: Initiative → Epic → Story → Task/Bug,
задачи и сроки, прогноз «Успеваем?», зависимости, стратегия, команда и материалы
для руководства. Без сервера и подписки.

**Открыть живую версию:** https://uchvru.github.io/roadmap/

В живой версии есть короткий видеообзор (демо, без звука) и полный каталог функций.

![Дорожная карта продукта](preview.png)

## Что это

Полноценный инструмент управления дорожной картой, собранный в один HTML-файл.
Открывается двойным кликом; базовая локальная работа не требует интернета и не
отправляет план на сервер. Сетевой обмен начинается только по действию пользователя,
например при переходе по внешней ссылке или настройке прямого подключения к Jira.
Подходит и продакт-оунеру, который ведет план и следит за сроками, и команде, и
руководству, которому нужен наглядный статус.

## Пять представлений одного плана

- **Сводка**, главное одним взглядом: в работе, просрочено, ближайшая дата, сигналы.
- **Список с диаграммой Ганта**, задачи слева, шкала справа, спринты и релизы.
- **Доска-канбан**, карточки по статусам, удобно на телефоне.
- **Успеваем?**, отдельная страница-ответ по каждой ключевой дате.
- **Презентация**, витрина роадмапа для показа руководству.

## Ключевые возможности

- **Честный прогноз сроков.** «Успеваем?» отвечает крупно и прямо: да с запасом, под вопросом или нет с опозданием. Считает по оценкам, зависимостям, отпускам и рабочим дням команды. Если данных мало, так и пишет, а не пугает мнимым срывом.
- **Песочница «Что если».** Подвигали темп, оценку или отпуск, сроки пересчитались сразу, реальный план при этом не трогается.
- **Зависимости** со стрелками, критический путь и подсветка нарушений.
- **Стратегический слой:** цели, инициативы, риски.
- **Команда** со ставками и занятостью, деньги, свой производственный календарь, свои поля.
- **Импорт из Jira** через CSV из вашей открытой сессии или через прямое подключение,
  которое пользователь включает и настраивает вручную.
- **Показ руководству:** витрина с выгрузкой на одну страницу PDF или в HTML.
- **Темная тема** и полностью русский интерфейс.

## Как устроено

- Один HTML-файл, около 3 МБ, работает офлайн.
- Ноль серверов, ноль подписок.
- Рабочие данные хранятся только у вас: в выбранном JSON-файле или локально в браузере.
- Публичный `app.html` работает только с вымышленным демонстрационным проектом.

## Как открыть

Нажмите на живую версию по ссылке выше или откройте [index.html](index.html) в
браузере. Онлайн-версия — демо: её можно свободно тестировать, но изменения не
сохраняются как рабочий проект.

## Приватность публичной сборки

- В репозитории нет рабочего `roadmap-data.json`.
- В `private/` публикуются только автономные HTML-контейнеры, содержимое которых
  зашифровано AES-256-GCM. Пароль не хранится в репозитории и передаётся отдельно.
- Названия и размеры файлов на GitHub Pages не являются секретными. Защищено именно
  содержимое; незашифрованные рабочие выгрузки в репозиторий добавлять нельзя.
- Jira-токены, настройки подключения и исходные `roadmap-data.json` не публикуются.
- Для демонстрации используются только встроенные вымышленные данные.
- Перед обновлением публикации демосборка проверяется отдельно от рабочего HTML.
